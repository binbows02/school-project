VCS Website revamped

@k9lol • 1 Person project

UX / UI Design • Figma • HTML / CSS • javaScript

Apr. 14, 2025

during a school project, we were all assigned to revamp multiple websites, and I got assigned to revamp a football related website

btw the website is open 24/7. heres the link to it: https://binbows02.github.io/PS2-respository/index.html

and heres the original: https://www.vcs-dh.nl/

UX / UI Designing

The problem

When most people visit the website, they were met with a not so well designed and user experience website: The website wanted to give out a “family” vibe.

Another problem was that it was difficult to sign in or check the roster (training & competitions) for the football club.

My task was to:

  • Handle & find a solution these problems
  • Redesign the website
  • Code the web design

Tackling the problem

In preperation, I've received:

  • Persona's
  • Debriefing


This way, I got to know the problems and made a User Needs (Prioritized by the MoSCoW method, which are “Must”, “Should”, “Could”, “Wont”) For myself to use while designing.

Other used techniques in preperation are:

  • Card sorting
  • Flowchart
  • Making a sitemap


Following that, I made a lo-fi and hi-fi prototype, which are both being tested many times by my classmates. Before the test I write down what I want them to do, During the tests I observe to see what they actually do. This way I can adjust things that wasn’t up to my expectations.

From them I've taken a lot of feedback and made a much more user experience-friendly website, whilst keeping an eye on the debriefing.

Coding

Use of codes

During this time, I had no prior knowledge about backend, databases and frameworks

This is why I've used JavaScript for logging in and the functions as being logged in as a football member (Such as a marked roster, or drop down menu settings)

Alongside, I've used HTML5 and CSS3 for structuring of the website.
